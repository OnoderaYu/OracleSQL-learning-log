## 💡 学習メモ・Tips

- `GROUP BY`の後に`HAVING`句を使うことで、集計結果を条件付きで絞り込める
- `JOIN`は結合の順序とON条件に注意しないと意図しない結果になる
- 実行計画（EXPLAIN）を使ってパフォーマンスをチェックする
